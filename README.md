# Parking Management System

A basic parking lot management and fee calculation console application developed with C#.

## Features
- Vehicle entry and exit system
- Parking capacity control
- Plate number tracking
- Hour-based fee calculation
- Console-based menu system

## Technologies Used
- C#
- .NET
- Console Application

## Project Purpose
This project was developed to improve my programming logic and gain experience with loops, conditional statements and real-life scenario based console applications in C#.
